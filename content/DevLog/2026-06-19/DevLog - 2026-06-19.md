# 2 Weeks of Development, and a new look

It's been 2 weeks since I started reviving this project. Using Cursor to help with the coding has been very helpful, and I was able to port the game to Unity, use some kitche art, reimplement the original gameplay of the demo, and add a mobile port.

## Same Gameplay Fresh Look

The Gameplay is the same, but I added some old kitchen art I made back then, and generated a simple model for the snake head. Still quite basic, but an improvement for sure. It was a bit of a hassle to get AI to generate such a simple mesh, so I think I will need human help for the art.

![[2026-06-18 17.10.03 1.gif]]
## GamePlay Upgrade: Android Swipe controls

Now we can swipe the screen to the side and vertically to control the Snake, which I think is quite intuitive and neat. We can also double-tap to go up.
